AI Transcript Copy

A bookmarklet that copies AI chat transcripts — ChatGPT, Gemini, NotebookLM, and Claude — to your clipboard as formatted rich text and markdown, ready to paste into Google Docs, Obsidian, or another AI.

Install (Desktop)

Drag this button to your bookmarks bar, or use the Copy button to copy the bookmarklet URL:

AI Transcript Copy
Make sure your bookmarks bar is visible first: Ctrl+Shift+B on Windows/Linux, Cmd+Shift+B on Mac.

Install (iOS)

Chrome's bookmark sync service drops bookmarks larger than ~6 KB, so the Standard bookmarklet won't sync to iOS Chrome. There are two options:

Option A — Sync the Basic bookmarklet via Chrome

The Basic bookmarklet is small enough to sync automatically. It handles all supported sites but skips deep research reports and Claude artifacts.

Drag to your bookmarks bar on desktop (it will sync to iOS), or use the Copy button:

AI Transcript Copy (Basic)

Option B — Manually install the Standard bookmarklet

The Standard bookmarklet can be installed manually on iOS with no size restriction. Use the Copy button below to copy the bookmarklet URL, then follow the steps for your browser.

Copy the bookmarklet URL to paste into a bookmark:

AI Transcript Copy

iOS Chrome

  1. Tap Copy URL above.
  2. Bookmark any page in Chrome (tap the share icon → Add to Bookmarks).
  3. Open Bookmarks, find the bookmark you just created, and tap Edit.
  4. Clear the URL field, long-press it, and tap Paste.
  5. Rename the bookmark to AI Transcript Copy and save.

iOS Safari

  1. Tap Copy URL above.
  2. Bookmark any page in Safari (tap the share icon → Add Bookmark). Save it.
  3. Open Bookmarks, find the bookmark, and tap Edit.
  4. Clear the URL field, long-press it, and tap Paste.
  5. Rename the bookmark to AI Transcript Copy and save.

Usage

  1. Open a conversation on a supported AI site.
  2. Click the AI Transcript Copy bookmark.
  3. The transcript is copied to your clipboard.
  4. Paste into Google Docs, Obsidian, Notion, etc.

Supported sites

  • ChatGPT (chatgpt.com)
  • Gemini (gemini.google.com)
  • NotebookLM (notebooklm.google.com)
  • Claude (claude.ai)

Limitations

  • ChatGPT Deep Research: Responses are rendered in a sandboxed cross-origin iframe that bookmarklet scripts cannot access. Workaround: use ChatGPT's built-in Export or Copy response button inside the deep research panel.
  • Claude Artifacts: Sidebar artifact content (generated code files, HTML previews, etc.) is rendered in an isolated iframe and cannot be copied. The surrounding chat response text is captured normally.
  • Gemini Deep Research Citations: Inline citation markers are stripped from the output because they contain no link or reference text — only a source index number.
  • Gemini Canvas / NotebookLM Studio: Canvas-rendered content may not be fully captured if it is displayed outside the standard message elements.
