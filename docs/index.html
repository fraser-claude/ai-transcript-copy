AI Transcript Copy

A bookmarklet that copies AI chat transcripts — ChatGPT, Gemini, NotebookLM, and Claude — to your clipboard as formatted rich text, ready to paste into a doc.

Install

Drag this button to your bookmarks bar:

AI Transcript Copy
Make sure your bookmarks bar is visible first: Ctrl+Shift+B on Windows/Linux, Cmd+Shift+B on Mac.

Usage

  1. Open a conversation on a supported AI site.
  2. Click the AI Transcript Copy bookmark.
  3. A dialog appears — choose which prompt to start copying from.
  4. Click OK. The transcript is now in your clipboard.
  5. Paste into Google Docs, Notion, Word, etc.

Supported sites

  • ChatGPT (chatgpt.com)
  • Gemini (gemini.google.com)
  • NotebookLM (notebooklm.google.com)
  • Claude (claude.ai)

Mobile

Drag-and-drop doesn't work on mobile. Instead:

  1. Bookmark any page.
  2. Edit that bookmark and replace the URL with the bookmarklet code from the GitHub repo.
  3. To run it, open the bookmarks menu and tap the bookmark — or type its name in the address bar.
